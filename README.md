# Stock Price Analysis Project

An advanced SQL project simulating a quantitative stock market analysis workflow. It demonstrates database schema design, data insertion, and data retrieval utilizing multiple relational table connections.

## SQL Techniques Demonstrated
* **INNER JOIN** & **LEFT JOIN**: Connecting active daily prices with corporate actions (dividends).
* **RIGHT JOIN**: Uncovering data gaps in corporate actions.
* **FULL OUTER JOIN**: Ensuring total data visibility using emulated UNION operations.
* **SELF JOIN**: Performing day-over-day price performance calculations (Today's Close vs. Yesterday's Close).
* **CROSS JOIN**: Generating a unique sector risk correlation matrix.
